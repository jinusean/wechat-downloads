# WeChat Downloads
## Automatically save downloads from WeChat

### Usage
1. .dmg and .app files found in `dist/`
2. App will run in background and accessed in menubar
<p align="center">
    <img src="./images/menubar.png"/>
</p>

### Development
```
pip install -r requirements.txt
python main.py
```
